# yFiles Gradle Plugin for Kotlin/JS

### Terms
- `C` - ordinal class
- `I` - ordinal interface
- `YC` - yFiles class
- `YI` - yFiles interface

### Planned support
- `C : YObject`
- `C : ... YI`
- `C : YC, ... YI`

### Planned errors
- `C : YObject, ... YI`
- `C : C, ... YI`
