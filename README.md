# Kotlin/JS declarations generator for yFiles

## Generation
* Run `./gradlew build`
* Check source folders
  * yFiles - `yfiles-kotlin/src/main/kotlin`
  * VSDX Export - `vsdx-kotlin/src/main/kotlin`
